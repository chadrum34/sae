# Android Fleet Update Channel

This public repository hosts signed Android Fleet browser-agent update metadata and patch archives.

- Customer databases, device IDs, APK files, USB drivers, XiaoWei installers, and signing private keys are never published here.
- Every update index is signed with Ed25519.
- Each client verifies its customer ID, package ID, archive size, and SHA256 before applying an update.
- Failed updates roll back and the installed version continues to start.

Do not edit files under `channels/` manually. They are generated by the Android Fleet release process.
